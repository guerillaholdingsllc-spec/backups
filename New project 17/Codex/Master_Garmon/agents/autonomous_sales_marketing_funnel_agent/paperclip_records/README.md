# Paperclip Records

This folder stores persistent, versioned records for the Autonomous Sales & Marketing Funnel Agent.

Required record types:

- Campaign records
- Creative and copy records
- Funnel asset records
- Landing page records
- Nurture sequence records
- KPI dashboard records
- A/B test records
- Daily cycle logs
- Weekly sprint plans
- Weekly KPI reports
- Compliance and risk records
- Data gap records

Each record should include a timestamp, source, linked asset or campaign IDs, rationale, version, status, and audit notes.
