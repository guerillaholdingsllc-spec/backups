# Campaign Cadence

Owner agent: Master Garmon
Child agent: Autonomous Sales & Marketing Funnel Agent
Created: 2026-05-15

## Always-On Operating Principle

The agent is expected to always have an active marketing objective, an active data collection objective, or an active optimization objective. When external access is missing, the agent must continue by producing campaign plans, asset drafts, analysis templates, dashboard schemas, test queues, and Paperclip records while flagging blocked live execution.

## Daily Cycle

1. Review active campaign metrics and alerts.
2. Collect channel, funnel, nurture, and test data.
3. Identify one winning signal and one weak signal.
4. Generate or update campaign assets.
5. Queue or recommend new content by platform.
6. Update KPI dashboard records.
7. Log all actions and data gaps in Paperclip.

## Weekly Cycle

1. Produce weekly KPI report.
2. Generate a new seven-day sprint plan.
3. Select campaign tests for hooks, offers, CTAs, creative, landing pages, and nurture sequences.
4. Pause or flag underperforming assets.
5. Double down on top-performing assets.
6. Update risk register.
7. Attach all decisions and reports to Paperclip.

## Monthly Cycle

1. Review channel ROI and funnel economics.
2. Update ICP and value proposition notes using collected evidence.
3. Refresh lead magnets and VSL angles.
4. Audit content library performance.
5. Produce monthly campaign archive.
6. Recommend budget allocation changes, if paid media access exists.

## Quarterly Cycle

1. Rebuild campaign map by funnel stage.
2. Review offer ladder performance.
3. Identify new market segments and dormant opportunities from stored data.
4. Refresh KPI targets.
5. Produce quarterly growth report.

## Yearly Cycle

1. Produce annual funnel performance review.
2. Archive all campaigns, assets, tests, and reports.
3. Identify highest-performing channels, offers, audiences, and content patterns.
4. Recommend next-year funnel and campaign strategy.
5. Update long-term risk and compliance register.
