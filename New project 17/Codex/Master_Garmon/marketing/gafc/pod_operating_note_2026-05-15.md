# GAFC Waitlist-First Local Drop Operating Note

Created: 2026-05-15
Owner: Master Garmon
Brand: Glocks & Fried Chicken (GAFC)

## Clarification

GAFC should operate as waitlist-first for now. Use POD-style mockups and Shopify demand capture to build a waiting list, then order selected merch and run a local drop after demand is proven.

## What This Means

- Sell the brand, character world, apparel designs, and local drop experience.
- Use Shopify as the main waitlist and demand-capture path.
- Do not imply there is stocked local inventory, same-day pickup, or retail availability before inventory is ordered and confirmed.
- Build campaign plans around waiting-list growth first, then local drop execution after demand is validated.
- Use early campaigns to collect demand data before committing to order quantities.

## Waitlist And Local Drop Messaging Rules

Approved language:

- Join the local drop waiting list
- Help decide what gets ordered
- First access goes to the list
- Local drop coming after demand is locked
- Waiting-list first
- Drop details coming soon
- Demand-tested designs

Avoid unless operationally true:

- In stock locally
- Pick up today
- Same-day fulfillment
- Available in store
- Retail location
- Local inventory

## Local Drop Path

After demand is validated, create a separate local drop plan covering:

- Local inventory strategy
- Drop or pickup location
- Local pickup process
- Local sales tax and compliance needs
- Point-of-sale setup
- In-person launch campaign
- Local partnership strategy
