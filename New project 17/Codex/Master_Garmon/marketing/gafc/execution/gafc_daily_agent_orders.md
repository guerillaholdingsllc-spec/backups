# GAFC Daily Agent Orders

Campaign: GAFC Local Drop Waiting List
CTA: Join the local drop waiting list and help decide what gets ordered.

## Morning Cycle

1. Agent 25 reviews the prior day's tracker.
2. Content engine selects the top product, hook, and objection.
3. Agents 01-24 receive updated content assignments.
4. Agents publish, queue, or prepare platform-ready content depending on available account permissions.
5. Agent 25 records data.

## Midday Cycle

1. Push product-vote content.
2. Ask for size demand.
3. Reply to comments with waitlist CTA.
4. Identify most requested item.

## Evening Cycle

1. Push urgency and first-access content.
2. Recap top requested products.
3. Ask followers to join before the order is planned.
4. Record all results.

## Daily Content Ratio

- 40% product reveal.
- 25% character/brand world.
- 20% waiting-list CTA.
- 10% product/size voting.
- 5% founder/build-in-public.

## Daily Success Question

Did today's content produce enough clear demand signals to help decide what gets ordered for the local drop?
